# Ideas (ID-)

Candidate solutions or approaches generated from sources through ideation. Ideas are the creative output of the pipeline — they represent possible improvements, solutions, or novel approaches.

## Cognitive Phases

Each idea should track which cognitive phase produced it:

| Phase | Description |
|-------|-------------|
| `consume` | Direct inspiration from reading/learning |
| `dream` | Free-association, no problem awareness |
| `daydream` | Semi-directed wandering |
| `contemplate` | Structured analytical thinking |
| `steal` | Pattern mapped from a different domain |
| `mate` | Combination of two or more existing ideas |
| `meta-learn` | Insight from analyzing what worked before |

## Scoring

Ideas can be scored on four dimensions (0-100):

- **Feasibility**: Can this actually be built/done?
- **Novelty**: Is this genuinely new?
- **Impact**: How much does it matter if it works?
- **Elegance**: Is the solution beautiful/simple?

## Adding an Idea

```bash
bun run ladder add idea --title "Your idea" --source SR-00001
```

## Upstream / Downstream

- **From**: Sources (SR-)
- **To**: Hypotheses (HY-)
