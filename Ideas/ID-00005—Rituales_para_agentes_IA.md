## Description

Implementar "rituales" para agentes de IA — procesos periódicos obligatorios de auto-evaluación, calibración y sincronización, inspirados en la función que los rituales religiosos cumplen para las sociedades:

- **Rezo diario** → Revisión diaria de objetivos y prioridades (¿sigo alineado con mi propósito?)
- **Misa semanal** → Sincronización semanal con otros agentes para compartir aprendizajes
- **Confesión** → Proceso de detección y reporte de errores propios sin penalización
- **Ayuno** → Periodos de restricción donde el agente opera con recursos limitados para detectar dependencias innecesarias
- **Peregrinación** → Exploración periódica fuera del dominio habitual para importar ideas de otros contextos

Estos rituales no son metáforas — son cron jobs con funciones concretas de mantenimiento sistémico.

Fase STEAL: tomar directamente la estructura de rituales religiosos y traducirla en procesos técnicos para agentes autónomos.
