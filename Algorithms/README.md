# Algorithms (AL-)

Proven methods for performing specific tasks. An algorithm in Ladder is any structured approach that has been validated through experimentation — a recipe that works.

## Scope

Algorithms here are broad — not just code. An algorithm might be:

- A process for reviewing pull requests
- A method for prioritizing feature requests
- A technique for debugging distributed systems
- A workflow for writing technical documentation
- An actual code algorithm

## Adding an Algorithm

```bash
bun run ladder add algorithm --title "Semantic similarity caching for REST APIs"
```

## Upstream / Downstream

- **Used by**: Experiments (EX-) as methodology
- **Validated by**: Results (RE-)
