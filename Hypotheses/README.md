# Hypotheses (HY-)

Testable predictions derived from ideas. A hypothesis takes a creative idea and turns it into something you can actually verify — "If X, then Y."

## Format

Every hypothesis must be:

1. **Testable** — you can design an experiment to validate it
2. **Specific** — you know what metric to measure
3. **Falsifiable** — you can define what failure looks like

## Adding a Hypothesis

```bash
bun run ladder add hypothesis --idea ID-00001 --title "Cache reduces p99 latency by 30%"
```

## Upstream / Downstream

- **From**: Ideas (ID-)
- **To**: Experiments (EX-)
